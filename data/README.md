# Data Directory

## Overview

This directory contains all datasets used in the ontology-aware anomaly detection project.

## Structure

- **`raw/`** - Original, unmodified dataset files
  - Place the downloaded Diabetes 130-US Hospitals CSV file here
  - Do not modify files in this directory

- **`processed/`** - Preprocessed and cleaned datasets
  - Generated by preprocessing notebooks
  - Ready for model training and analysis

## Dataset Information

### Diabetes 130-US Hospitals Dataset (1999-2008)

**Download**: [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/datasets/Diabetes+130-US+hospitals+for+years+1999-2008)

**Description**: 
- 10 years (1999-2008) of clinical care at 130 US hospitals
- Approximately 100,000 patient records
- 50+ features including patient demographics, diagnoses, medications, and lab procedures

**Key Features**:
- Patient demographics (age, gender, race)
- Admission details (admission type, source, length of stay)
- Medical codes (ICD-9 diagnoses, procedures)
- Medications (insulin, diabetic medications)
- Lab tests (HbA1c, glucose levels)
- Readmission status

**Citation**:
Beata Strack, Jonathan P. DeShazo, Chris Gennings, Juan L. Olmo, Sebastian Ventura, Krzysztof J. Cios, and John N. Clore, "Impact of HbA1c Measurement on Hospital Readmission Rates: Analysis of 70,000 Clinical Database Patient Records," BioMed Research International, vol. 2014, Article ID 781670, 11 pages, 2014.

## Usage Notes

- Keep raw data unchanged for reproducibility
- All preprocessing scripts should read from `raw/` and write to `processed/`
- Large data files are excluded from version control (see `.gitignore`)
